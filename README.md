# MacroPad Firmware

CircuitPython firmware for Adafruit MacroPad RP2040

## Features

- 12-key macro execution with RGB feedback
- Profile switching via encoder
- Multiple macro types: Once, Hold, Toggle  
- Loop support with conditional exits
- Emergency stop all (encoder button)
- Custom LED colors per macro

## Installation

1. Install CircuitPython 8.x on MacroPad
2. Copy all .py files to CIRCUITPY drive
3. Create data/ structure or use Configurator
4. Reset device (Ctrl+D or Ctrl+Shift+R)

## Usage

- **Keys 0-11**: Execute macros
- **Encoder Rotate**: Switch profiles
- **Encoder Press**: Emergency stop all
